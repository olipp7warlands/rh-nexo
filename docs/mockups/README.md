# Mockups — especificación visual

Estos prototipos de alta fidelidad son la **fuente de verdad de UI/UX**. No son funcionales
(no persisten datos); su función es definir el aspecto, el layout, los copys y los estados de
cada pantalla. Ábrelos en el navegador.

- `nucleo-rrhh.html` — el Core de RRHH completo (Inicio, Empleados, Organigrama, Ausencias,
  Onboarding, Desempeño, Nómina, Documentos, Reclutamiento). 
- `vitae-mockup.html` — el ATS (VITAE) en profundidad.

Al construir cada módulo, replica lo que se ve aquí con los componentes de `@nucleo/ui`.
